# Icon 图标

`CIcon` 用于渲染 `packages/font/*.js` 注入的 SVG symbol。

## 示例

```vue
<c-icon name="icon-shijian" />
```

## API

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `name` | symbol id，必填 | `String` | - |
| `className` | 额外 class | `String` | - |
| `width` | svg 宽度 | `Number` | - |
| `height` | svg 高度 | `Number` | - |

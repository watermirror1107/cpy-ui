# 组件清单

`cpy-ui` 里的组件都放在 `packages` 目录，每个组件通常由一个 `index.vue` 实现。

| 组件 | 说明 |
| --- | --- |
| `CBanner` | 页面顶部标题栏，支持返回图标和自定义内容 |
| `CButton` | ant-design-vue Button 增强，支持图标、提示、业务状态色 |
| `CIcon` | SVG symbol 图标组件 |
| `CCard` | 内容卡片容器，支持标题、更多操作、标题插槽 |
| `CCardItem` | 卡片内的标题和值展示行 |
| `CInput` | 带浮动 label 的输入框 |
| `CInputNumber` | 带加减按钮和单位的数字输入 |
| `CRadio` / `CRadioGroup` | 卡片式单选按钮组 |
| `CDuration` | 月份/年限时长选择 |
| `CRangePicker` | 时间范围选择，带快捷时间按钮 |
| `CSearchSelect` | 异步搜索下拉，支持分页加载和多选 |
| `CForm` | 配置式表单 |
| `CTable` | 表格封装，支持异步数据、分页、轮询、操作栏 |
| `CTableForm` | 表格查询条件表单 |
| `CTableBtn` | 表格操作按钮 |
| `CBtnWrap` | 表格操作按钮自动折叠容器 |
| `CTabs` | ant-design-vue Tabs 简化封装 |
| `CTap` | 统计卡片式筛选组件 |
| `CEllipsis` | 多行省略文本，溢出时 popover 展示全文 |
| `CEditableCell` | 可编辑文本单元格 |
